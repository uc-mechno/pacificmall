## 概要
wordpress、gulpなどの学習用に作成をしたwordpressのスターターテンプレートです。

## 特徴
- gulpでejsをhtmlにコンパイル
- gulpでscssをcssにコンパイル
- gulpでjavascriptをES6→ES5にバンドル
- gulpで画像をtinyPNGで圧縮
- gulpで画像をWebPに出力
- gulpでハッシュの追加
- gulpでブラウザの自動リロード
